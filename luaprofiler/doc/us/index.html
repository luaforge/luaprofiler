Overview

LuaProfiler is a time profiler designed to help finding bottlenecks on your Lua program.

If you use LuaProfiler into your program, it will generate a log of all your function calls and their respective running times. You can use this log to generate a summary of the functions your program uses, along with how long it stayed in each of them, or you can import the log into a spreadsheet for further analysis.

LuaProfiler is free software and uses the same license as Lua 5.0.

Status

Current version is 2.0 It was developed for Lua 5.0.2, 4.0.1 and 3.2.

Version 2.0 follows the package model for Lua 5.1 (see section Installation for more details).

Download

LuaProfiler can be downloaded from the LuaForge page. If you are using LuaBinaries Release 2 a Windows pre-compiled version of LuaProfiler can also be found at the same LuaForge page.
